description: Create New Objective
---

Create a new objective "$ARGUMENTS":

1. Generate objective ID (OBJ-XXX, increment from highest in pool)
2. Gather details:
   - What needs to be done?
   - Why does it matter?
   - Done when? (specific criteria)
   - Size estimate (XS/S/M/L/XL)
   - Autonomy level required
3. Add to `.flow/objectives/pool.md`
4. If urgent or fits obvious stream, suggest immediate assignment
5. Report back with objective ID and next steps
