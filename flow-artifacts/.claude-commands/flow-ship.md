description: Ship Completed Objective
---

Complete the current objective:

1. Identify current objective from:
   - Git branch name
   - Recent work context
   - Ask if unclear
2. Verify completion criteria are met
3. Update objective with:
   - Status: completed
   - Completion date
   - Summary of what was done
   - Learnings captured
4. Update stream file:
   - Move objective from active to completed
   - Add learnings to stream learnings log
5. Update metrics if significant
6. Check if stream is now complete (all success criteria met)
7. If stream complete, trigger reflection
8. Report back with completion summary

If work isn't done:
- List remaining items
- Ask what's blocking completion
