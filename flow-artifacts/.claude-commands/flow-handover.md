description: Generate Session Handover
---

Create a handover document for the current session:

1. Summarize what was accomplished this session
2. Document current state:
   - Git branch and status
   - Uncommitted changes
   - Active objectives and progress
   - PRs created/updated
3. List decisions made with rationale
4. Identify concrete next steps
5. Note any blockers or open questions
6. Update:
   - `.context/SESSION_HANDOVER.md` with immediate context
   - `.flow/streams/active/[current].md` with stream state
   - Any objective work logs
7. Present handover summary for review
